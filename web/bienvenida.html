Bienvenidos al módulo de Alertas y Monitore de Transportes PowerDSD

La empresa de transportes PowerDSD tiene la necesidad de implementar controles y monitoreo para asegurar el desplazamiento de personas en sus distintas agencias de transportes. Para este efecto, se requiere por un lado, identificar a sus clientes quienes dicen ser, consultando la base de datos de Reniec para esta validación. Por otro lado, se requiere conocer si estos clientes tienen alguna requisitoria policial vigente para alertar a la policía y a su tripulación para que se tome las contingencias del caso. En este último se consultará la base de datos de requisitorias de la policía.
